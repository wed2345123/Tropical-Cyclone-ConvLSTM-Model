# Tropical-Cyclone-ConvLSTM-Model
A ConvLSTM model for tropical cyclone track prediction.
## About This Project
This repository contains the source code for a deep learning model designed to simulate and predict typhoon paths.

## File Structure
```
.
└── Model.py                  # Main model script
```

## Getting Started

### Prerequisites
- Python 3.8+
- PyTorch
- NumPy
- Matplotlib

### Installation
We recommend using [Anaconda](https://www.anaconda.com/) to manage your environment.

1.  **Create and activate a conda environment:**
    ```bash
    conda create -n pytorch python=3.8
    conda activate pytorch
    ```

2.  **Install the required packages:**
    ```bash
    pip install torch numpy matplotlib
    ```

## Usage
To run the model, you will need to provide your own dataset.

1.  **Modify the script** to load your data and model.
2.  **Execute the script:**
    ```bash
    python "Model1 - 副本.py"
    ```
